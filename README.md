#  Telecom Customer Churn Analysis

##  Project Overview
This project analyzes telecom customer data to identify factors that contribute to customer churn.

##  Objectives
- Identify key factors affecting churn
- Analyze customer behavior
- Predict churn patterns
- Provide business insights

##  Dataset
- Source: Telecom dataset
- Features: Customer demographics, usage, charges, tenure

##  Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Analysis

##  Key Insights
- Customers with high monthly charges are more likely to churn
- Contract type plays a major role
- Long-term customers churn less
